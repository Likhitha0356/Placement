# See the latest changes

If you don't see any updates in the app, do this:

## 1. Stop the app
In the terminal where `npm start` is running, press **Ctrl+C** (or Cmd+C on Mac) to stop it.

## 2. Start again
```bash
cd /Users/likki/Desktop/placements/placement-portal
npm start
```

## 3. Hard refresh the browser
- **Windows/Linux:** `Ctrl + Shift + R` or `Ctrl + F5`
- **Mac:** `Cmd + Shift + R`

Or: open DevTools (F12) → right‑click the refresh button → **Empty Cache and Hard Reload**.

## 4. What you should see

- **Navbar:** Green **"Updated"** badge next to "Placement Portal". When you toggle the theme (sun/moon), the navbar turns dark gray in dark mode.
- **Sidebar:** In light mode it’s white; in dark mode it’s dark. The current page is highlighted in indigo.
- **Cards:** On any page (Dashboard, Reports, etc.), cards and backgrounds switch with the theme (dark = dark gray cards).
- **AI Agent:** Ask different questions (e.g. "interview tips", "resume", "skills") — you should get different answers each time.
- **Mock Interview:** After login, open Mock Interview. You should see a **large avatar** on the left (interviewer) and **your camera** on the right with a confidence % that updates when the camera is on.

If you still don’t see these, say which of these steps you did and what you see instead.
